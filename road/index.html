    Product Roadmap

    Core Platform

    UI Designer v2

    2026-Q1

    Refactoring the low-code interface for a simpler and more intuitive process digitization experience.

    UI for Test Generation

    2026-Q2

    Development of a user interface that allows for on-the-fly test generation across any environment.

    Integration with Flink Technology for Streaming Operations on Kafka Queues

    2026-Q3

    In order to convert analytical operations from batch processes to real-time streaming operations, we are implementing Flink technology, which enables complex operations on streaming data from Kafka queues.

    Codebase Update

    2026-Q4

    Updating the automation codebase to keep pace with technological advancements.

    UI Runtime Automation v2

    Refactoring the operational management interface for running digitalized processes.

    Wizard and Automation Startup

    2027-Q1

    Development of a user interface that enables the deployment and startup of automations.

    Modular Automations

    2027-Q2

    Development of pre-configured automations with specific tasks that can be used in a modular way to construct processes, thereby reducing implementation time.

    Adding Event-Driven Architecture

    2027-Q4

    Introduction of an event-driven architecture for automation operation to complement the current polling architecture.

    Load JARs at Runtime

    Introducing the functionality to update automation code at runtime, allowing patches or upgrades to be applied without interrupting process execution.

    Industries & Use Cases

    Digital Twin Energy Efficiency in Smart Buildings

    2025-Q4

    Module for modeling consumption, efficiency improvements, and energy forecasting.

    Digital Twin Predictive Maintenance in Manufacturing Plants

    2026-Q2

    Module for synchronization with SCADA/MES systems and real-time scenario simulations for Predictive Maintenance and quality assurance in production.

    Digital Twin for Cybersecurity Operational Processes

    2026-Q4

    Module for making cybersecurity technologies interoperable to enhance the efficiency of Security Operations Centers (SOC).

    AI & Automation Intelligence

    Expand the anomaly detection packages with more effective algorithms

    2026-Q1

    Implementation of new anomaly detection algorithms, in addition to the existing models.

    Integration with agenticAI systems

    2026-Q3

    Integrating agenticAI systems into the digitalization process pipeline, both from the process design perspective and the maintenance of data analysis algorithms.

    Increase the speed and scalability of root cause algorithms

    2026-Q4

    Using Flink technology to scale root cause algorithms for faster and more dynamic use.
