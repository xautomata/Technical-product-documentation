# Product Roadmap

## Core Platform

**UI Designer v2**

- **Theme:** Refactoring the low-code interface for a simpler and more intuitive process digitization experience.
- **Timing:** Q1 2026

**UI for Test Generation**

- **Theme:** Development of a user interface that allows for on-the-fly test generation across any environment.
- **Timing:** Q2 2026

**Integration with Flink Technology for Streaming Operations on Kafka Queues**

- **Theme:** In order to convert analytical operations from batch processes to real-time streaming operations, we are implementing Flink technology, which enables complex operations on streaming data from Kafka queues.
- **Timing:** Q2 2026

**Codebase Update**

- **Theme:** Updating the automation codebase to keep pace with technological advancements.
- **Timing:** Q4 2026

**UI Runtime Automation v2**

- **Theme:** Refactoring the operational management interface for running digitalized processes.
- **Timing:** Q4 2026

**Wizard and Automation Startup**

- **Theme:** Development of a user interface that enables the deployment and startup of automations.
- **Timing:** Q1 2027

**Modular Automations**

- **Theme:** Development of pre-configured automations with specific tasks that can be used in a modular way to construct processes, thereby reducing implementation time.
- **Timing:** Q2 2027

**Adding Event-Driven Architecture**

- **Theme:** Introduction of an event-driven architecture for automation operation to complement the current polling architecture.
- **Timing:** Q4 2027

**Load JARs at Runtime**

- **Theme:** Introducing the functionality to update automation code at runtime, allowing patches or upgrades to be applied without interrupting process execution.
- **Timing:** Q4 2027

## Industries & Use Cases

**Digital Twin Energy Efficiency in Smart Buildings**

- **Theme:** Module for modeling consumption, efficiency improvements, and energy forecasting.
- **Timing:** Q4 2025

**Digital Twin Predictive Maintenance in Manufacturing Plants**

- **Theme:** Module for synchronization with SCADA/MES systems and real-time scenario simulations for Predictive Maintenance and quality assurance in production.
- **Timing:** Q2 2026

**Digital Twin for Cybersecurity Operational Processes**

- **Theme:** Module for making cybersecurity technologies interoperable to enhance the efficiency of Security Operations Centers (SOC).
- **Timing:** Q4 2026

## AI & Automation Intelligence

**Expand the anomaly detection packages with more effective algorithms**

- **Theme:** Implementation of new anomaly detection algorithms, in addition to the existing models.
- **Timing:** Q1 2026

**Integration with agenticAI systems**

- **Theme:** Integrating agenticAI systems into the digitalization process pipeline, both from the process design perspective and the maintenance of data analysis algorithms.
- **Timing:** Q3 2026

**Increase the speed and scalability of root cause algorithms**

- **Theme:** Using Flink technology to scale root cause algorithms for faster and more dynamic use.
- **Timing:** Q4 2026
